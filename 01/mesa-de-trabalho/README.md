Os alunos deverão se reunir em dupla, e buscar e corrigir os erros presentes no projeto. Existem errors de sintaxe, links quebrados e type error.

- [ ] Corrigir errors de link quebrado
- [ ] Corrigir SintaxeError
- [ ] Corrigir TypeError
